# Floating Time

A simple, elegant, and always-on-top desktop clock application created with Java Swing. It provides a clean, transparent, and movable digital clock that stays visible over all other windows.

## Features

- **Always-on-Top:** The clock remains visible over other applications.
- **Transparent Background:** Integrates seamlessly with your desktop wallpaper for a clean look.
- **Movable:** Easily drag and drop the clock to any position on your screen.
- **Simple Controls:** Intuitive mouse controls for moving and closing the application.
- **Lightweight:** A single Java file with no external dependencies.

## Prerequisites

To compile and run this application, you need to have the **Java Development Kit (JDK)** installed on your system (version 8 or newer is recommended).

## How to Run

1.  **Save the Code:**
    Save the code into a file named `Floating_Timr.java`.

2.  **Open a Terminal:**
    Open your command prompt (Windows) or terminal (macOS/Linux).

3.  **Navigate to the File Location:**
    Use the `cd` command to navigate to the directory where you saved the file.
    ```bash
    cd path/to/your/directory
    ```

4.  **Compile the Code:**
    ```bash
    javac Floating_Timr.java
    ```

5.  **Run the Application:**
    ```bash
    java Floating_Timr
    ```
The clock will appear on your screen.

## Controls

- **Move Window:** **Left-click and drag** the clock to the desired position.
- **Close Application:** **Right-click** anywhere on the clock.
